# 06 — Architecture

Especificación técnica completa de la plataforma: diagrama de servicios, tecnologías por capa,
esquema de base de datos, flujos de datos entre componentes, y configuración del VPS único.

Incluye decisiones de diseño justificadas, alternativas descartadas, y restricciones de recursos
(RAM, CPU, almacenamiento) bajo presupuesto €0 con stack 100% open-source.
