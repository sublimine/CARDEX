# 04 — Extraction Pipeline

Define las estrategias E1-E12 de extracción de catálogo de vehículos: desde APIs públicas y RSS/Atom
hasta sitemap crawling, JSON-LD parsing, y feeds estructurados de portales.

Cada estrategia documenta: tipo de fuente, tecnología requerida, legalidad, tasa de cobertura esperada,
complejidad de implementación, y dependencias de infraestructura.

Restricción absoluta: ninguna estrategia usa evasión de controles, fingerprint spoofing, ni proxies residenciales.
