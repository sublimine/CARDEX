# 05 — Quality Pipeline

Especificación de los validadores V01-V20 del pipeline de calidad de datos: validación de campos
obligatorios, normalización de precios/monedas, deduplicación por VIN, verificación de imágenes,
y scoring de completitud de listing.

Objetivo: zero-error en datos publicados al índice. Cada validador tiene umbral de rechazo,
acción correctiva, y métrica de monitorización asociada.
