## Principios editoriales

1. **No inventar datos.** Si una cifra no se conoce con certeza, se marca como `[PV: OrderOfMagnitude]` (Pendiente Verificación) con la fuente oficial donde consultar.
2. **Citar fuentes.** Toda afirmación cuantitativa referencia su origen (organismo, publicación, fecha).
3. **Fechar las referencias.** El mercado automotriz cambia; una estadística sin fecha puede ser engañosa.
4. **Distinguir hecho verificable de estimación.** Las estimaciones se marcan `(est.)` y se explica la metodología.
5. **Honestidad epistémica sobre exactitud.** Una tabla con algunos `[PV]` es más valiosa que una tabla con números inventados.

## Convención de marcadores

| Marcador | Significado |
|---|---|
| `[PV: ~N]` | Pendiente Verificación — orden de magnitud estimado N |
| `(est.)` | Estimación con metodología documentada |
| `(ACEA 2024)` | Fuente y año de la cifra |
| `[fuente oficial: URL]` | Fuente a consultar para verificación |
