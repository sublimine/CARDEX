# 03 — Discovery System

Documentación completa del sistema de descubrimiento de dealers: 15 familias de fuentes,
clasificadas por tipo (OEM, portal, registro, mapas, social, etc.) y cobertura por país.

Cada familia tiene: descripción, fuentes concretas, método de acceso legal, datos disponibles,
volumen estimado, y referencias a estrategias de extracción (ver 04_EXTRACTION_PIPELINE).

Subdirectorio `families/` contiene un archivo por familia con detalle operativo.
