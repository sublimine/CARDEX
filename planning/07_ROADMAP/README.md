# 07 — Roadmap

Plan de ejecución por fases con criterios de cierre cuantitativos: dealers descubiertos,
vehículos indexados, países activos, uptime, latencia de actualización.

Cada fase tiene: objetivo medible, dependencias de fases previas, riesgos identificados,
y criterio de aceptación binario (pass/fail). Sin estimaciones temporales subjetivas.
